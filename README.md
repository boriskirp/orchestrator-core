# Orchestrator Core
Центральний вузол активації сенсів для бізнес-методології "Оркестратор".

## Функціонал
- **Синхронізація:** Автоматичне оновлення знань у локальних файлах.
- **Активація:** Динамічне підсвічування ключових бізнес-термінів.
- **Потік:** Побудова цілісної вирви розуміння для клієнта.
